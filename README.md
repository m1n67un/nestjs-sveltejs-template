# template-mingty-ns

> 위 프로젝트는 svelte와 nest 템플릿 프로젝트입니다. by mingty

> - svelte 3.35.0
> - rollup 2.41.2
> - @nestjs/cli 7.5.6

### 모듈 설치
``` sh
$ npm i
```

### 빌드
``` sh
$ npm run build
```

### 프로젝트 실행
``` sh
$ npm start
```

### 프로젝트 실행 - watch mod
``` sh
$ npm run start:dev
```

### 프로젝트 실행 - debug mod
```sh
$ npm run start:debug
```
